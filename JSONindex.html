JSON

JavaScript Object Notation

Def: JavaScript Object Notation (JSON) dan mengonversi nilai menjadi JSON dan format untuk berbagi data.

Lightweight data-interchange Format JSON adalah format yang ringan dan mempermudah kita dapat membagi, menyimpan dan bekerja dengan data, basic utk berbagi data server dan user, tetapi kita harus menentukan terlebih dahulu struktur data yang tepat utk project kita apakah JSON, XML Atau AJAX.

    DATA TYPES:

  • NUMBERS: No difference between integer and floats
  • STRING: String of Unicode characters. Uses double quotes
  • Boolean: True or False
  • Array: Ordered list of 0 or more values
  • OBJECT: Unordered collection of key/value pairs
  • Null:Empty value
  • Contoh JSON Object
  • Ada lagi JSON API, JSON PHP, JSON ARRAY, XML

    JSON Syntax Rules:

  • Uses key/value pairs -{"nama":"Ihsandroid"}
  • So when we define it adjacent script decent syntax is always think camelCase
  • Uses double quotes around KEY and Value
  • Must use the specified data type
  • File type is .json
  • MIME type is "Application/json"
  • Object bisa Array, bisa dipisahkan dengan tanda comma
